## 2. 글씨스타일

1.  ‘** 글자 **’ :  **볼드 체** 
2. ‘* 글자*’: *italic*
3. ‘_글씨_’ : *기울어진 글씨*
4. ‘~~글씨~~’: ~~글씨에 선긋기~~
5.  ‘``` ': 소스코드 작성 

```jsx
console.log(`sourceCode`)
```
